🚀 Scenery Addons Browser v3.5.8

This update focuses on a redesigned update experience, improved reliability, and clearer user control when new versions are available.

✨ What’s New

  • Interactive update dialog
    Review what’s new before updating and decide whether to update now or later.
  • Update available indicator
    Postponed updates remain visible and can be reopened anytime.
  • Improved GSX profile detection
    More accurate results from flightsim.to searches.
  • Developer update mode
    Force the update dialog for testing purposes.

🛠 Improvements

  • Updates are now fully user-driven (no silent updates).
  • Refined splash screen and update flow.
  • Clearer status messages and improved logging.

🐞 Bug Fixes

  • Fixed updates applying without user confirmation.
  • Fixed update dialog appearing behind splash or main window.
  • Minor UI glitches and stability improvements.
Thank you for using Scenery Addons Browser ✈️
Your feedback helps shape future updates.
